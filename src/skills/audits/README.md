# Skill audits

This directory stores resumable audit state and human-readable batch reports for
the skill catalog.

- `skill-audit-state.json` — generated by `scripts/audit_skills.py --resume`.
  Skills marked `audited` or `done` are skipped on the next resumable batch
  unless `--force` is used.
- `*.md` reports — generated with `scripts/audit_skills.py --report` or written
  during manual category sweeps. Generated report filenames include timestamps
  to preserve earlier batches.

Drain pending static audits in one run:

```bash
python3 scripts/audit_skills.py --batch-size 15 --resume --until-complete --report
```

Do not put secrets, credentials, copied private docs, or raw model transcripts
in audit reports.
