# Vault Map

## Active surfaces

- `Home.md` — command center / dashboard.
- `000 Inbox/` — transient capture.
- `001 Projects/Threads/` — ongoing themes and synthesis.
- `001 Projects/Works/` — concrete outputs.
- `002 Areas/` — responsibilities and admin.
- `003 Resources/` — media, documentation, templates, periodic notes, QuickAdd support.
- `004 Archives/` — retired systems, redirects, deletion candidates.

## Maintenance surfaces

- `.obsidian/community-plugins.json` — plugin enablement list.
- `.obsidian/appearance.json` — visual posture only.
- `.obsidian/workspaces/*.json` — saved layouts.
- `004 Archives/Redirects.md` — moved-note map.
- `004 Archives/Deletion Candidates.md` — safe-delete queue.

## Safety rules

- Archive before delete.
- Keep plugin/token data out of git.
- Maintain redirects for renamed/moved notes.
- Prefer one edit path per task: config, content, or validation.
