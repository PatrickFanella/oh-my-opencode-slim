# Statistical Distance

Turn cliché defaults into fresher equivalents while preserving emotional function.

## Purpose

Keep same emotional vector. Move from statistical center to statistical edge.

## When to Use

- First instinct feels generic
- Element seen too many times
- You need differentiation without breaking function

## When Not to Use

- You haven’t identified the element’s real function yet
- You’re trying to make every element surprising

## Workflow

1. **Capture instinctive element**
2. **Name functional core** (stakes/pressure/sympathy/etc.)
3. **Enumerate center options** (3–5 common defaults)
4. **Push to edge** using one move:
   - adjacent substitution
   - complication layering
   - ironic inversion
   - category jump
   - specificity injection
5. **Test fertility + core retention**

## Output Checklist

- [ ] Functional need clearly stated
- [ ] Statistical center options listed
- [ ] 5+ edge alternatives generated
- [ ] Selected edge passes fertility test
- [ ] Emotional core preserved
- [ ] Distance calibrated (fresh, not alien)

## Guardrails

- Don’t push everything to edge
- Don’t trade clarity for novelty
- Don’t keep label while retaining same dynamic
- Don’t keep edge options that generate no new story possibilities
