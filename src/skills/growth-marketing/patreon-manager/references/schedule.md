# Patreon promotion schedule

## Daily

- Publish `changemaker`.
- Convert one line into X snippet.
- Share one process note to Instagram Stories.
- If there is a visual/process moment, record or repurpose a short Reel/TikTok.

## Weekly

- Tuesday: Patreon deep dive.
- Thursday: poll, draft, or roadmap note.
- Sunday: supporter recap.
- 2 X threads/week.
- 1 Instagram carousel/week.
- 1+ Reel/week.
- 1+ TikTok/week.
- 1 companion note when a `subcult.tv` post ships.
- Metrics check: choose next week’s strongest angle.

## Monthly

- Roadmap + patron input post.
- Supporter roll call / thank patrons.
- Public monthly project recap.
- Update Patreon pitch based on performance.

## First 30 days

1. Publish pinned welcome post.
2. Add Patreon links to all social bios.
3. Prepare first 30-day calendar.
4. Create reusable templates.
5. Define project list for roadmap posts.
