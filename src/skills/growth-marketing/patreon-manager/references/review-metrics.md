# Review metrics

## Weekly review

Track:

- Patreon posts published vs planned.
- Public posts published vs planned.
- Best performing X post/thread.
- Best performing Instagram Story/Reel/carousel.
- Best performing TikTok.
- Link clicks, joins, comments/replies, saves/shares.
- Best source angle: shipped, learned, failed, decision, preview, roadmap.

Output:

- Keep / stop / try next week.
- One strongest angle to repeat.
- One weak channel/task to reduce or simplify.

## Monthly review

Track:

- Patron count change.
- Paid conversion signals.
- Top 5 posts by meaningful engagement.
- CTA performance.
- Which projects drove interest.
- Whether cadence felt sustainable.

Update:

- `Monthly Promotion Review.md`.
- Patreon pitch if needed.
- Repeating tasks only if user approves recurrence changes.
