# SEO Pro

Unified SEO workflow for technical SEO, AI search visibility, schema markup, and programmatic SEO.

## Use When

- SEO audit, indexing, metadata, rankings, crawlability, or traffic drop.
- Schema.org / JSON-LD / Google rich results.
- AI SEO / answer engine optimization / LLM citations.
- Programmatic SEO templates and data-driven landing pages.

## Routing Inside This Skill

- **Technical/on-page SEO:** audit crawlability, metadata, canonicals, sitemap, robots, performance, internal links.
- **Schema:** add or validate JSON-LD for Organization, Article, Product, FAQ, Breadcrumb, SoftwareApplication, etc.
- **AI SEO:** make pages answer-ready, citeable, structured, and source-rich.
- **Programmatic SEO:** design templates, data model, URL patterns, uniqueness rules, and quality gates.

## Workflow

1. Identify page/site type and business goal.
2. Check current state: URLs, CMS/framework, Search Console if available, schema, indexability.
3. Choose lane: audit, schema, AI visibility, pSEO, or combined.
4. Produce prioritized fixes with effort/impact.
5. For implementation, include exact metadata/schema/template changes.
6. Validate with available crawlers, rich-result tests, and generated page QA.

## Output Checklist

- [ ] Primary SEO problem and target pages identified.
- [ ] Fixes prioritized by impact and confidence.
- [ ] Schema is accurate to visible page content.
- [ ] pSEO pages have uniqueness and thin-content safeguards.
- [ ] AI-search recommendations include quotable answers and source structure.
- [ ] Validation commands/tools listed.
