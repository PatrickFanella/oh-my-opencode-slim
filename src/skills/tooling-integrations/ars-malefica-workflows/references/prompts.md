# Prompt Snippets

## Capture prompts

- “What kind of note is this: journal, resource, thread, work, or documentation?”
- “Give me title, medium/type, and source URL if it’s a resource.”
- “Give me title, core question, and related note links if it’s a thread.”
- “Give me title, publish target, and optional dates if it’s a work.”

## Routing reminder

```text
If it is lived experience -> Journal
If it is a source -> Resource
If it is a theme -> Thread
If it is an artifact -> Work
If it teaches the vault -> Documentation
```
